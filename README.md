# Lab_Web_Django

This repo is COMM Automatic System Tool website that has following functions:

  - [x] [Show PC information](#Dashboard) 
  - [X] [Add tasks to PC](#Taskmanager)
  - [x] [Show tasks status and taks result](#Result)
